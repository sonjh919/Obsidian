## 🌈 init
+ 현재 디렉토리를 기준으로 **Git  저장소**가 생성된다.
+ 내부에 `.git` 디렉토리가 생기며, 여기에는 Git 버전 관리를 위한 **메타 정보**가 담겨 있다.
```git
git init
```
