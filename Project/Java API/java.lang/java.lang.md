## 🌈 java.lang
+ java 프로그래밍에 필요한 가장 기본적인 클래스들이 모여있는 패키지이다.
+ 모든 클래스의 최상위 클래스인 **Object** 클래스가 있다.
+ java.lang은 Compiler가 자동으로 추가해주기 때문에 java API 내에서 유일하게 import하지 않고 쓸 수 있다.
