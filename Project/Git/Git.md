
## 🌈 Git이란?
깃(Git)은 2005년에 개발된 **분산 버전관리 시스템**으로, 컴퓨터 파일의 **변경사항을 추적**하고 **여러명의 사용자들 간 파일에 대한 작업을 조율**하는데 사용된다.
