### 🌈 Set
+ 집합과 관련된 것들을 쉽게 처리하기 위해 만든 자료형이다.
+ **중복을 허용하지 않는다.**
+ **순서가 없다.(unordered)**

#### 🧶 하위 클래스
+ HashSet
+ LinkedHashSet
+ TreeSet

> 💡 **HashSet, LinkedHashSet, TreeSet**
HashSet : 순서가 없다.
LinkedHashSet : 값을 입력한 순서대로 정렬한다.
TreeSet : 값을 오름차순으로 정렬해 저장한다.
