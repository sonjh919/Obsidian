## 🌈 산술 연산자

- **NULL이 포함된 사칙연산의 결과는 NULL이다.**

| 연산자 | 의미          |
| ------ | ------------- |
| ()     | 우선순위 조정 |
| *      | 곱하기        |
| /      | 나누기        |
| +      | 더하기        |
| -      | 빼기          |

## 🌈 합성 연산자
- 문자, COL명 간 연결 시 사용한다.

```sql
SELECT COL1 || ' ' || 'SQLD' FROM SAMPLE;
```
