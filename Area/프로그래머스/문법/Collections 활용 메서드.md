#코딩테스트 #Java 


+ [[Collection]]의 구현체인 Collections에 포함되어 있는 메서드를 활용할 수 있다.
```java
import java.util.Collections;
```


### 배열 정렬하기
```java
Collections.sort(list);
```

### 배열 뒤집기
```java
Collections.reverse(list);
```

### 배열 최대/최소 구하기
```java
int max = Collections.max(list);

int min = Collections.min(list);
```
